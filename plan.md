# Neo-Brutalism 重构计划

## 设计基线
- 定义色板：高饱和粉/黄/蓝/绿 + 对比黑白；制定描边(2-4px)、偏移阴影、贴纸元素规则。
- 建立全局样式文件 `css/brutal.css`：重置、字体栈（粗黑+等宽备选）、按钮/卡片/导航/布局组件、动效(120-200ms, hover 漂浮旋转, active 压下)。
- 准备噪点/条纹背景和贴纸角标的 CSS 方案（纯 CSS/小 SVG）。

## 页面改造顺序
1) `index.html`：首页导航、英雄区、主行动召唤、特色块、公告替换为“最新动态”，加入色块栅格与偏移阴影。
2) `about.html`：俱乐部使命/成员/设备资源，用色块信息卡 + 时间轴徽章。
3) `history.html`：里程碑时间轴，使用左/右错落色块和虚线连接；清理文本格式。
4) `cgzs.html`：成果卡片化，添加贴纸感标签与外链按钮，保证图片边框与阴影。
5) `hdyx.html`：文件列表改为粗野卡片网格，加载态/错误态统一样式，面包屑贴纸化。
6) `dh.html`：导航目录改为彩色标签瀑布流，搜索条强化；首字母分组用色块头标。
7) `qq-groups.html`：群卡片改为粗描边+飘带，复制提示改用浮层贴纸；强化筛选按钮。
8) `vpn.html`：工具页做成卡片表单，修正乱码，给出转换示例。
9) `notice.html` & `403.html`：维护提示页统一视觉，减少乱码。

## 收尾
- 自查可访问性：对比度、focus 样式。
- 移动端断点检查。
- 资源与路径核对，移除废旧内联样式。
- 更新 README/说明若有必要。
